#### Setup env for batch transfers

```bash
npm i
touch .env
```

Fill `.env` file

```
PRIVKEY = …
CONTRACT_ADDR = …

INFURA_APIKEY = …

NETWORK_NAME = kovan | mainnet

BATCH_SIZE = 200
GAS_PRICE = 1e9
GAS_LIMIT = 12e6

STARTING_NONCE =
```
